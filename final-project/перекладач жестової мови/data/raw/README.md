
У цій папці має бути:
- необроблені відео жестів
- зображення або короткі кліпи
- початкові матеріали, отримані без обробки

### Технології:
- формат файлів: `.mp4`, `.jpg`, `.png`


### Cтуктура набору даних

Підпапки - назви жестів (класи моделі).

   - Жест1
     - картинка1.jpg
     - картинка2.jpg
   - Жест2
     - video1.mp4
     - image1.jpg
   ...
