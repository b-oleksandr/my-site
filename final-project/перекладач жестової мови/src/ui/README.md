 У цій папці має бути:
- файл `ui.py` для графічного інтерфейсу
- адаптери для конвертації кадрів OpenCV у формат, що показує GUI
- віджети (кнопки, текстові поля, вікно програми)

### Технології:
- **PySimpleGUI** (найпростіший варіант GUI)
- **OpenCV** — відображення зображення у GUI у форматі PNG
