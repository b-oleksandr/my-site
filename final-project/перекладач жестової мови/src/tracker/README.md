У цій папці має бути:
- модуль `mediapipe_hand.py` для визначення координат руки
- модуль `preprocessing.py` для нормалізації та обробки даних
- формування вектора ознак для моделі

### Технології:
- **MediaPipe Hands** — детекція та трекінг руки
- **NumPy** — для обробки масивів і нормалізації
- **OpenCV** — для базової роботи з кадрами
