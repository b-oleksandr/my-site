У цій папці має бути:
- модуль для запису відео та фото жестів на вебкамеру (з простим віконним інтерфейсом) за допомогою OpenCV (зберегати у data/raw)
- модуль для обчислення точок MediaPipe та збереження у `.csv` (дані з папки data/raw, зберегти csv у data/processed)

### Технології:
- **MediaPipe Hands** — отримання структурованих даних
- **NumPy** — зберігання наборів даних
- **OpenCV** — зйомка відео з камери
