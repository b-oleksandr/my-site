У цій папці має бути:
- фінальний датасет для тренування моделі
- файли у форматах `.csv`
- дані з мітками жестів

### Технології:
- **NumPy**
- **Pandas** (опціонально)


### Процес обробки:

 - Відео розбивається на кадри та зберігаються у картинки (за допомогою ffmpeg)
 - Картинки жестів беруться без змін
 - Для кожного зображення запускається модель MediaPipe Hand Landmarker
 - На виході ми отримуємо вектор координат які зберігаються у CSV файл для подальшого тренування класифікаційної моделі

Клас (жест) визначається назвою папки в якій знаходиться відео або картинка.
